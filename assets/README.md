# Assets

This folder contains project images and logos.

## Files

- **obelisk-logo.jpg** - Main project logo (square, used in README header)
- **obelisk-og.jpg** - Open Graph image for social media sharing (1200x630px recommended)

## Usage

The logo is automatically displayed in the main README.md. The og-image can be configured in:
- GitHub repository settings (Social preview)
- Or referenced in meta tags if you have a website
